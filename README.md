# Sistem Pakar Diagnosa Penyakit

Aplikasi sistem pakar berbasis web menggunakan PHP Native dan MySQL
untuk membantu proses diagnosa penyakit berdasarkan gejala.

## Cara Menjalankan
1. Simpan project di folder htdocs
2. Jalankan Apache & MySQL (XAMPP)
3. Import database
4. Buka di browser
